Deskripsi Pekerjaan

Kami mencari Senior Software Engineer dengan pengalaman minimal 5 tahun di bidang backend development. Kandidat harus memiliki pengetahuan mendalam tentang sistem terdistribusi, microservices architecture, dan best practices dalam software engineering.

Sebagai bagian dari tim engineering kami, Anda akan bertanggung jawab untuk:

  • Mengembangkan dan memelihara aplikasi backend yang scalable
  • Melakukan code review dan mentoring untuk junior engineers
  • Merancang dan mengimplementasikan solusi teknis yang robust
  • Berkolaborasi dengan tim product dan design untuk menghasilkan produk berkualitas
  • Melakukan optimization dan performance tuning pada sistem yang ada

Kualifikasi

Persyaratan Wajib:

  • Pengalaman minimal 5 tahun di backend development
  • Mahir menggunakan Node.js, Python, atau Java
  • Pemahaman mendalam tentang SQL dan NoSQL databases
  • Familiar dengan Git dan version control system
  • Pengalaman bekerja dengan microservices dan API development
  • Kemampuan communication dan teamwork yang baik

Keahlian yang Diharapkan:

  • Pengalaman dengan Docker dan Kubernetes
  • Knowledge tentang AWS atau cloud services lainnya
  • Familiar dengan CI/CD pipeline dan DevOps practices
  • Pengalaman dengan React.js atau frontend frameworks lainnya
  • Pemahaman tentang system design dan scalability
